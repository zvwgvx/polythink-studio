# PolyThink Studio

A sophisticated web application for reviewing and editing datasets.

## Features
- **Item-Level Editing**: Edit role, thinking, and content fields directly.
- **Refined UI**: Dark mode with a professional aesthetic.
- **Smart Interactions**: Auto-save prompts and custom confirmation modals.

## Setup

### Backend
```bash
cd backend
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python main.py
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
